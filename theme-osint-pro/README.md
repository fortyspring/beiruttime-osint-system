# OSINT Pro Theme

ثيم WordPress متخصص مصمم للعمل مع إضافة OSINT Pro، يدعم الاشتراكات المدفوعة، الإعلانات، تتبع Google، ومتجاوب مع جميع الأجهزة.

## المميزات الرئيسية

### 🔐 نظام الاشتراكات المدفوعة
- تكامل كامل مع WooCommerce Subscriptions
- قفل المحتوى المميز للمشتركين فقط
- صفحات اشتراك مخصصة
- إدارة الصلاحيات بناءً على حالة الاشتراك

### 📺 نظام الإعلانات
- أماكن إعلانية متعددة (أعلى الصفحة، داخل المحتوى، الشريط الجانبي، الفوتر)
- دعم Google AdSense وأي شبكات إعلانية أخرى
- تتبع نقرات الإعلانات عبر Google Analytics
- إدارة سهلة من خلال تخصيص القالب

### 📊 تتبع Google
- تكامل مدمج مع Google Analytics 4
- تتبع أحداث مخصص (نقرات الإعلانات، المشاركات الاجتماعية)
- إعداد سهل من خلال تخصيص القالب
- دعم Universal Analytics و GA4

### 📱 متجاوب مع الهواتف
- تصميم Responsive يعمل على جميع الأجهزة
- قائمة هاتف محمولة
- صور متجاوبة
- تحسين سرعة التحميل للأجهزة المحمولة

### 🌐 دعم اللغة العربية
- اتجاه RTL مدمج
- خطوط عربية جميلة (Tajawal)
- ترجمة كاملة للعربية
- دعم كامل للغات الأخرى

### 🔗 تكامل مع OSINT Pro
- كشف تلقائي عن الإضافة
- عرض تقارير OSINT بشكل جميل
- محتوى مميز للتقارير الحصرية
- أدوات مشاركة اجتماعية

## التثبيت

### الطريقة 1: الرفع اليدوي
1. قم بتنزيل مجلد `theme-osint-pro`
2. ارفع المجلد إلى `/wp-content/themes/`
3. فعل القالب من لوحة تحكم WordPress

### الطريقة 2: FTP
1. اتصل بخادمك عبر FTP
2. اذهب إلى `/wp-content/themes/`
3. ارفع مجلد `theme-osint-pro`
4. فعل القالب من لوحة التحكم

## الإعداد الأولي

### 1. إعداد الاشتراكات
1. ثبت وأفعّل WooCommerce
2. ثبت وأفعّل WooCommerce Subscriptions (اختياري)
3. أنشئ منتجات اشتراك في WooCommerce
4. حدد صفحة الاشتراكات باستخدام قالب "صفحة الاشتراكات"

### 2. إعداد الإعلانات
1. اذهب إلى **مظهر > تخصيص**
2. اختر قسم **الإعلانات**
3. ألصق أكواد الإعلانات في الأماكن المطلوبة:
   - أعلى الصفحة
   - داخل المحتوى
   - الشريط الجانبي
   - أسفل الصفحة

### 3. إعداد Google Analytics
1. احصل على معرف التتبع من Google Analytics
2. اذهب إلى **مظهر > تخصيص**
3. اختر قسم **تتبع Google**
4. أدخل معرف التتبع (مثل: G-XXXXXX أو UA-XXXXX-Y)

### 4. إنشاء محتوى مميز
1. عند إنشاء أو تحرير مقال، أضف خاصية مخصصة:
   - الاسم: `_osint_premium_content`
   - القيمة: `1`
2. استخدم الكود القصير `[osint_premium]` حول المحتوى المميز

## الأكواد القصيرة (Shortcodes)

### زر الاشتراك
```
[osint_subscribe text="اشترك الآن" plan="premium"]
```

### محتوى مميز
```
[osint_premium locked="true"]
هذا المحتوى متاح للمشتركين فقط
[/osint_premium]
```

### الإعلانات
```
[osint_ad position="content"]
```

## تخصيص المحتوى المميز

### تحديد محتوى كـ "مميز"
```php
update_post_meta($post_id, '_osint_premium_content', '1');
```

### التحقق من حالة الاشتراك
```php
if (osint_pro_is_premium_user()) {
    // المستخدم مشترك مميز
} else {
    // المستخدم عادي
}
```

### قفل المحتوى برمجياً
```php
$content = osint_pro_lock_content($content, true);
```
